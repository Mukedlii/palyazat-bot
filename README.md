# 🇭🇺 Magyar Pályázat Figyelő Bot

Minden nap 14:00-kor értesít az aktuális magyar pályázatokról és támogatásokról!

## Telepítés

1. npm install
2. node bot.js

## Parancsok

- /start – Indítás, kategória választás  
- /palyazatok – Azonnali pályázat lista
- /stop – Leiratkozás napi értesítőről
- /help – Súgó

## Hogyan működik?

- Minden nap 14:00-kor automatikusan összegyűjti a friss pályázatokat
- Személyre szabva küldi a felhasználók kategóriája szerint
- Adatforrások: palyazatok.org, palyazat.gov.hu
